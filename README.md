# INR to BDT Converter 🪙

A simple, logic-based Python script that converts Indian Rupees (INR) to Bangladeshi Taka (BDT) using a defined exchange rate.

## 📝 About the Project
This project was created as part of my journey into *Python Programming*. My goal was to practice:
* Taking and handling user input.
* Implementing mathematical logic in code.
* Managing basic error handling (ensuring the user enters a valid number).

## 🚀 How to Run
To run this project on your local machine, follow these steps:

1. *Prerequisites:* Ensure you have Python installed.
2. *Download:* Save converter.py to your computer.
3. *Execution:* Open your terminal or Command Prompt (CMD) and run:
   ```bash
   python converter.py
